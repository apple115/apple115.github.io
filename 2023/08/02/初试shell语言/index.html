初始shell语言

什么是shell

Shell 脚本编写是一种使用 shell 命令在 Unix/Linux 系统中自动执行任务的方法。
shell 是一个命令行解释器，用于执行脚本中的命令。
Shell 脚本可以执行各种操作，例如文件操作、程序执行和打印文本。

什么是shell脚本

Shell 脚本是用脚本语言编写的，例如 Bash (Bourne Again SHell),zsh,fish。
Shell 脚本是逐行执行的一系列命令。
Shell 脚本可以包含变量、控制结构（例如循环和条件）和函数。

从打印’hello world!’ 开始

新建一个’helloworld.sh’文件
写下

#!/usr/bin/env bash
echo "hello world!"

#!告诉系统其后路径所指定的程序即是解释此脚本文件的 Shell 程序。

  1. 使用bash运行
> bash helloworld.sh
  1. 使用chmod添加权限
> chmod +x helloworld.sh
> ./helloworld.sh

我们就会在shell上打印出hello world!
.指的是当前的目录
./helloworld.sh
只是选择当前的目录下的文件
这时shell 会假设为一个可执行程序

实际上shell 中运行

> echo "hello world!"
hello world!

shell 脚本相当于把在shell上运行命令放在一起
